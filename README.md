# Калькулятор риска ЗППП во времени

Интерактивная иллюстративная модель (React + Vite + Recharts). Публикуется бесплатно на GitHub Pages.

> ⚠️ Это любительский калькулятор, а не медицинский инструмент. Цифры — грубые оценки и не предсказывают личный риск. Решения о тестировании/лечении принимать с врачом.

---

## Как опубликовать на GitHub Pages (с нуля)

1. **Создай репозиторий** на GitHub (название любое, например `sti-risk-model`). Можно публичный или приватный — Pages работает с обоими.

2. **Залей файлы из этой папки** в репозиторий. Два способа:

   **A. Через сайт (без терминала):** на странице репозитория → `Add file` → `Upload files` → перетащи всё содержимое этой папки (вместе с папками `src` и `.github`) → `Commit changes`.
   > Внимание: при перетаскивании через браузер скрытая папка `.github` иногда не загружается. Если её не видно — загрузи файл `.github/workflows/deploy.yml` отдельно (создай через `Add file → Create new file` и впиши путь `.github/workflows/deploy.yml`).

   **B. Через git (терминал):**
   ```bash
   git init
   git add .
   git commit -m "init"
   git branch -M main
   git remote add origin https://github.com/ТВОЙ_ЛОГИН/ИМЯ_РЕПО.git
   git push -u origin main
   ```

3. **Включи GitHub Pages:** в репозитории → `Settings` → `Pages` → в разделе `Build and deployment` → `Source` выбери **GitHub Actions**.

4. Готово. После каждого `push` в ветку `main` workflow сам соберёт сайт и опубликует его. Ссылка появится в `Settings → Pages` (вид `https://ТВОЙ_ЛОГИН.github.io/ИМЯ_РЕПО/`). Первая сборка занимает 1–2 минуты; статус виден во вкладке `Actions`.

## Локальная разработка

Нужен установленный Node.js (версия 18+).

```bash
npm install      # поставить зависимости (один раз)
npm run dev      # запустить локально, откроется на http://localhost:5173
npm run build    # собрать продакшен-версию в папку dist
npm run preview  # посмотреть собранную версию локально
```

Весь код калькулятора — в одном файле `src/App.jsx`. Правишь его (сам или через Claude), делаешь `git push` — сайт обновляется автоматически.

## Структура

```
.
├─ index.html                  # точка входа
├─ package.json                # зависимости и скрипты
├─ vite.config.js              # base: "./" — чтобы работало на любом подпути Pages
├─ src/
│  ├─ main.jsx                 # монтирует React
│  └─ App.jsx                  # ← весь калькулятор здесь
└─ .github/workflows/deploy.yml # автодеплой на GitHub Pages
```
